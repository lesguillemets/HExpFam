    data Uniform a Source #

    Uniform distribution. Range is inclusive. Make a distribution either by i) explicitly giving the range and p, or ii) fromRange function.

    >>> unif0 = Uniform (1,10) 0.1 :: Uniform Int
>>> f unif0 5
0.1
>>> f unif0 100
0.0
>>> fromRange (1,10) == unif0
True

    >>> unif1 = Uniform (0,10) 0.1 :: Uniform Double
>>> f unif1 5.1
0.1
>>> unif1 == fromRange (0,10)
True
