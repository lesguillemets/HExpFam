class IntegrateConfig i where Source #

Information about how to calculate the integral. Domain i is usually Double, but we can define integration of some function f : ℂ → ℝ on a line, in which case Domain i might be Complex. We would like to use this to calculate the expected value over discrete random distribution too, for which Domain i shall be an iteration over the support.

data IntegrateDouble Source #

Integrate from _lower to _upper, using boxes whose width is _dx.

>>> take 5 . show $ integrate (IntegrateDouble 0 (pi/2) 0.001) $ sin
"1.000"

data IntegrateDiscrete a Source #

For Discrete distributions, use all the values with equal weight, 1.

>>> data Foo = Foo | Bar | Baz | Foz deriving (Enum, Bounded)
>>> integrate (IntegrateDiscrete :: IntegrateDiscrete Foo) (fromIntegral . fromEnum)
6.0
